# MyQuickLauncher
ランチャーアプリ


## 概要
キーボードの操作のみで登録済みのアイテムを起動するランチャーです。
自分の環境で動くことを前提にしているので設定機能などはありません。。

## 使い方
無変換でアプリを表示します。

登録済みアイテムのクリック、登録済みアイテムのに該当するキーの押下でアプリ起動。それだけです

アイテムの編集はドラッグドロップ、もしくかコンテキストメニューから行います。
